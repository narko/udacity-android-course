In order to use the key that you obtained from themoviedb.org please follow this process:

1. Open [USER_HOME]/.gradle/gradle.properties. Create the file gradle.properties if it doesn't exist yet.
2. Add the following line: MyTheMovieDBApiToken="XXXXX". Replace "XXXXX" with your key.
